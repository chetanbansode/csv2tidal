# Tips & Tricks

- Use `Fast` mode for large casual transfers.
- Use `Accurate` mode for playlists where album/version correctness matters.
- Keep JSON reports enabled when testing new playlists.
- Name your CSV files clearly, because the filename becomes the default playlist name.
- If a playlist is especially important, start with a smaller test CSV first.

> [!TIP]
> Start with a smaller playlist first when you are testing a new transfer mode or a new CSV export.
